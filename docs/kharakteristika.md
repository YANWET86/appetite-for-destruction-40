# Характеристика проекта

**Тема:** Юбилейный бокс-сет Guns N' Roses «Appetite for Destruction — 40th Anniversary». Продающий лендинг от вымышленного лейбла Black Frost Records.

**Концепция.** Лендинг построен как **цифровой коллекционный артефакт**, а не интернет-магазин. Эстетика — винтажная пластинка 1987 года: глубокий чёрный фон, состаренные латунные акценты, кровавый красный для целевых кнопок, плёночное зерно поверх всего сайта. Шрифтовая пара Cinzel (заголовки в духе постеров эпохи) + Inter (основной текст) + Special Elite (даты и штампы, печатная машинка) задаёт «документальный» характер.

**Структура.** 6 содержательных блоков по методичке: герой-экран с обратным отсчётом до релиза 21 июля 2027 года, состав бокса (6 предметов: винил 180g, артбук, литография, реплика билета, сертификат, NFC-чип со скрытым треком), выгоды покупателя (4 карточки: один из 5000, именной сертификат, бесплатная доставка, гарантия выкупа), CTA-панель с ценой и счётчиком остатков, социальные доказательства (статистика, упоминания прессы, цитаты), FAQ и форма связи.

**Визуальные средства.** Семнадцать анимаций уровня Awwwards: прелоадер с прогресс-баром, побуквенная печать заголовка, плавный смарт-скролл Lenis, появление блоков с размытием при попадании в зону видимости, параллакс hero-визуала, магнитные кнопки призыва к действию, 3D-наклон карточек товара, переворачивающийся таймер обратного отсчёта, нарастающий счётчик статистики, бесконечная лента логотипов прессы, тикающий каждые 30 секунд счётчик остатков, плавный аккордеон FAQ, валидация формы с эффектом тряски при ошибке.

**Технологии.** Vanilla HTML + CSS + JavaScript без сборщиков и фреймворков. Все библиотеки подключены по CDN: GSAP 3 с плагинами ScrollTrigger и SplitText, Lenis для смарт-скролла, vanilla-tilt.js для 3D-эффектов. Шрифты — Google Fonts, фото — Unsplash, иконки — inline SVG Lucide. Деплой — GitHub Pages.

**Доступность.** Все анимации длиннее 0.3 секунды отключаются при `prefers-reduced-motion: reduce` (соответствие WCAG 2.3.3). Магнитные кнопки и tilt-карточки отключаются на тач-устройствах. Цели тач-нажатия — не менее 44×44 px (WCAG 2.5.5).

**Производительность.** Все анимации используют исключительно GPU-ускоренные свойства (transform/opacity). Целевые показатели Lighthouse: Performance ≥ 95, Accessibility ≥ 90.

**Контент.** Текст написан без рекламной слащавости — короткие предложения, глаголы в настоящем времени, цифры и факты. Никаких «эксклюзивно», «торопитесь», «уникальная возможность». Доверие выстраивается через детали: имена авторов архивных фотографий, тип бумаги артбука, метод нарезки винила, технология аутентификации.
