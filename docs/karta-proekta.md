# Карточка проекта

**Тема проекта:**
Юбилейный коллекционный бокс-сет рок-альбома Guns N' Roses «Appetite for Destruction — 40th Anniversary Edition».

**Тип проекта:**
Автономный одностраничный продающий лендинг (landing page).

**Категория представленной информации:**
Презентация коллекционного товара ограниченного тиража + приём предзаказов.

**Целевое действие пользователя:**
Оформить предзаказ коллекционного бокс-сета по цене 24 990 ₽ (тираж 5000 нумерованных экземпляров).

**Объект продвижения:**
Вымышленный лейбл-издатель — Black Frost Records (студия лимитированных переизданий, выпускающая физический бокс к 40-летию альбома 1987 года).

**Цена / тираж:**
24 990 ₽ за экземпляр. Глобальный тираж — 5 000 нумерованных боксов.

**Целевая аудитория:**
Коллекционеры музыки 30–55 лет, поклонники классического хард-рока, ценители физических носителей (винил, артбуки). Готовы платить за культурный артефакт.

**Адрес сайта:**
`https://yanwet86.github.io/appetite-for-destruction-40/`
(публикуется через GitHub Pages после деплоя).
